Free e-commerce system plugin

== Description ==

WP-OliveCart4 is a free e-commerce plugin made in Japan.
With WP-OliveCart4, you can easily to e-commerce shop on Wordpress.

WP-OliveCart4はWordPressにショッピングカート機能を追加する日本語対応、日本製プラグインです。
今あるサイトをシンプル＆簡単にショッピングサイトに。

== Installation ==

= Minimum Requirements =

* PHP version 7.1 or greater (PHP 7.1 or greater is recommended)
* MySQL version 5.6 or greater (MySQL 5.6 or greater is recommended)
* WordPress 5.3+

= Automatic installation =
 1. Go to the *Plugins* menu and click *Add New*.
 2. Search for *wp-olivecart*.
 3. Click *Install Now* next to the *wp-olivecart* plugin.
 4. Activate the plugin.

= MANUAL INSTALLATION =

1. Alternatively, download the plugin and upload the entire e-shop folder to the /wp-content/plugins/ directory.
2. Activate the plugin.
